# WinXTended Project
The Windows XP modification project, created, directed, and maintained by WinUpgrade, oriented on providing the most commonly used program set for everyday-use and with some fancy wallpapers :P

## NO OPTIONAL DRIVERS INCLUDED AND THEY'LL NEVER BE. IF YOU WANT TO HAVE SUCH, PLEASE CONSIDER INTEGRATE THEM USING NLITE OR BY INSTALLING DRIVERS FOR YOUR SPECIFIC HW AFTER THE MAIN OS INSTALLATION!!!!

## What included:
- Useful everyday software (such as Supermium, Notepad++, TotalCMD, etc.)
- Some fancy wallpapers from Win Vista
- Automated software install on 1st boot (done with RunOnce)
- Royale theme by default
- All the other contents that Windows XP has by default 

## Known issues (planned to fix in further releases):
- No drivers for specific hardware included
- Wrong product key included for automatization (you do have enter the product key yourself)
- If you have found any other bugs or issues, submit an Issue commit
